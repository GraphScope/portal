## Graphy Website

## Quick Start

### Perpare

- install node.js : https://nodejs.org/en
- install pnpm : https://pnpm.io/installation#using-npm `npm install -g pnpm`

### Install Dependencies

```bash
pnpm install
```

### Run Website

```bash
npm run start
```

visit http://localhost:8000/dataset/
